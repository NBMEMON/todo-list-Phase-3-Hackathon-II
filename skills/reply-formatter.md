# Reply Formatter Skills

This directory contains the skill definitions for the reply formatter agent.

## Skills

### 1. Response Formatting
- Generate natural language responses
- Format responses appropriately for different intents
- Ensure responses are helpful and clear

### 2. Emoji Integration
- Add relevant emojis to enhance responses
- Use emojis appropriately based on context
- Maintain professional tone while using emojis

### 3. Language Detection
- Detect language of user input
- Respond in the same language when possible
- Handle mixed-language inputs appropriately

### 4. Contextual Awareness
- Generate responses that acknowledge conversation history
- Refer back to previous interactions when relevant
- Maintain consistent personality and tone

### 5. Error Response Generation
- Create helpful error messages
- Guide users toward successful interactions
- Maintain positive tone even during error conditions
