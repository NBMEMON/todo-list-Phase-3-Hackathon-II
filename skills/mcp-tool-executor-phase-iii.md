# MCP Tool Executor Phase III Skills

This directory contains the skill definitions for the MCP tool executor agent.

## Skills

### 1. Database Operations
- Execute database queries and updates
- Handle task creation, reading, updating, and deletion (CRUD)
- Manage user-specific data isolation

### 2. JWT Validation
- Validate JSON Web Tokens for authentication
- Ensure user authorization for requested operations
- Handle token expiration and refresh

### 3. API Coordination
- Interface with backend API endpoints
- Format requests according to API specifications
- Handle API responses and errors appropriately

### 4. Transaction Management
- Ensure data consistency during operations
- Handle rollbacks when operations fail
- Manage concurrent access safely

### 5. Error Handling
- Catch and handle database errors
- Provide meaningful error messages
- Maintain system stability during failures
